# go.md — 에이전트 작업 지시서

> 이 파일을 수정하면 에이전트들이 읽고 아래 작업을 자동으로 수행합니다.
> `npm start`를 실행하면 에이전트가 이 파일을 읽고 토큰이 소진될 때까지 작업합니다.

---

## 프로젝트 개요

간단한 Todo 앱을 만들어주세요.

## 기술 스택

- HTML / CSS / JavaScript (바닐라)
- 별도 프레임워크 없이 순수 JS로 구현

## 태스크 목록

아래 태스크를 순서대로 완료하세요. 각 태스크 완료 후 "태스크명 완료"라고 출력하세요.

### 태스크 1: 프로젝트 구조 생성
- `output/` 폴더를 생성하세요
- `output/index.html` 파일을 생성하세요 (기본 HTML 뼈대)
- `output/style.css` 파일을 생성하세요 (기본 스타일)
- `output/app.js` 파일을 생성하세요 (기본 JS 뼈대)
- 완료 시 출력: "태스크1 완료"

### 태스크 2: Todo 앱 UI 구현
- `output/index.html`에 Todo 입력폼과 목록 UI를 구현하세요
- `output/style.css`에 깔끔한 스타일을 적용하세요
- 완료 시 출력: "태스크2 완료"

### 태스크 3: Todo 앱 로직 구현
- `output/app.js`에 다음 기능을 구현하세요:
  - Todo 추가
  - Todo 완료 체크/해제
  - Todo 삭제
  - localStorage 저장
- 완료 시 출력: "태스크3 완료"

### 태스크 4: 최종 검토 및 README 작성
- 구현된 파일들을 검토하고 버그가 있으면 수정하세요
- `output/README.md`를 작성하세요 (사용법, 기능 설명)
- 완료 시 출력: "태스크4 완료"

---

## 주의사항

- 각 파일은 실제로 생성/수정해야 합니다
- 작업 완료 키워드를 반드시 출력해야 다음 단계로 넘어갑니다
- 현재 작업 디렉토리 기준으로 파일을 생성하세요
