# 자율 에이전트 실행 작업지시서

## 목표
LangChain/LangGraph 기반 에이전트가 이 파일을 읽고 작업을 수행한다.
컨텍스트 사용량이 80% 이상이면 현재 작업 내역과 남은 작업을 자동 기록한 뒤,
새 세션(새 창)에서 다시 이 파일을 읽고 이어서 작업한다.

## 규칙
- 태스크는 `###` 헤더 단위로 작성한다.
- 완료된 태스크는 자동 생성 영역의 체크리스트로 관리된다.
- 자동 생성 영역(`<!-- AUTO-GENERATED ... -->` 아래)은 수동 수정하지 않는다.

## 태스크 목록

### 태스크 1: 요구사항 확인 및 실행 준비
- 프로젝트 설정(.env, package.json, src 구성)을 확인한다.
- 실행 전에 필요한 CLI(claude/gemini/codex) 사용 가능 여부를 점검한다.
- 완료 시 "태스크1 완료"를 출력한다.

### 태스크 2: 기능 구현
- 현재 태스크 요구사항에 맞게 파일을 수정한다.
- 필요하면 테스트 또는 검증 명령을 실행한다.
- 완료 시 "태스크2 완료"를 출력한다.

### 태스크 3: 결과 정리
- 변경 파일과 핵심 결과를 간단히 요약한다.
- 남은 이슈/추가 작업이 있으면 다음 태스크로 넘긴다.
- 완료 시 "태스크3 완료"를 출력한다.
